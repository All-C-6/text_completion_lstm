# Проект LSTM модели для дополнения текстов.

#### Модель дополняет текст с 75% до 100% (генерирует еще примерно треть токенов от поданных ей)

## Структура проекта:

- configs/: папка для конфигурации проекта (пути, гиперпараметры и тд)
- src/: папка с самим кодом работы модели, вся логика находится тут
    - common_utils.py: настройка логирования и умная загрузка файла датасета
    - data_preprocess.py: все от чтения файла датасета до создания объекта DataLoader (3 шт: train, val и test)
    - lstm_model.py: код самой модели, класс с его методами
    - eval_transformer_pipeline.py: валидация предобученной GPT2 модели с выводом результатов.
    - train.py: тренировка модели LSTM на train DataLoader
    - eval.py: валидация LSTM модели на val DataLoader
    - next_token_dataset.py: не используется
